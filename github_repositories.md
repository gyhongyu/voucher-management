# gyhongyu GitHub 倉庫總結清單 (GitHub Repositories Summary)

此文件摘要整理了帳戶 `gyhongyu` 旗下的 GitHub 倉庫，包含公開、私有及 Fork 倉庫。為便於後續 AI 代理或人工查閱，已將倉庫依「專案屬性」進行分類整理。

## 1. AI ＆ 大模型原生應用 (AI & LLM Native Projects)

### 🏢 [AI-XLSX](https://github.com/gyhongyu/AI-XLSX)
*   **權限**：🔒 Private (私有)
*   **定位**：基於本地開源大模型的 XLSX 文件知識擷取系統。
*   **特色**：與一般 RAG 不同，它直接從電子表格中提取真實數據進行查詢、計算、排序和推理，保證回答嚴謹度並杜絕 AI 幻覺。

### 📎 [files](https://github.com/gyhongyu/files)
*   **權限**：🌐 Public (公開)
*   **定位**：公共檔案中轉站。
*   **特色**：專門為 AI 應用程序在交互過程中需要上傳、下載臨時文件所設計的公共中轉倉庫。

### ⌨️ [OfficeJS-RAG](https://github.com/gyhongyu/OfficeJS-RAG)
*   **權限**：🌐 Public (公開)
*   **定位**：OfficeJS 與 檢索增強生成 (RAG) 的整合專案。

### 🤖 [gemini-cli](https://github.com/gyhongyu/gemini-cli) (Fork)
*   **權限**：🌐 Public (公開 / 分支)
*   **定位**：終端命令列 AI 代理工具，將 Google Gemini 的能力直接帶入終端。

### 🌐 [browser-use](https://github.com/gyhongyu/browser-use) (Fork)
*   **權限**：🌐 Public (公開 / 分支)
*   **定位**：讓 AI 代理人能夠直接點擊、輸入並自動化操作網頁的控制工具。

### 📂 [Multi-company-DMC](https://github.com/gyhongyu/Multi-company-DMC)
